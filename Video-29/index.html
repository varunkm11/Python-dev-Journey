Nested List in Python

Nested lists are lists that contain other lists as their elements. They are useful for representing complex data structures such as matrices, grids, or trees.

Example:
matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

In the example above, matrix is a nested list representing a 3x3 grid. Each inner list is a row of the matrix.

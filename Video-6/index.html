Python Variables

In Python, variables are used to store data. You donâ€™t need to declare the data type explicitly; Python automatically detects it based on the value assigned.

Example:


# Assigning values to variables
x = 5           # Integer
name = "Alice"  # String
pi = 3.14       # Float
is_active = True  # Boolean

# Printing variables
print(x)
print(name)
print(pi)
print(is_active)

Variable names must begin with a letter or underscore, and they are case-sensitive.

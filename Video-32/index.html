Tuples in Python

A tuple is an immutable, ordered collection of items in Python. Tuples are similar to lists, but unlike lists, they cannot be changed (they are immutable) after creation.

Key Features of Tuples

  • Ordered: The items have a defined order and that order will not change.
  • Immutable: You cannot add, remove, or change items after the tuple is created.
  • Allow duplicate values.
  • Can contain elements of different data types (integers, strings, etc.).

Creating a Tuple

my_tuple = (1, "apple", 3.14)

Accessing Tuple Elements

print(my_tuple[1])  # Output: apple

Why Use Tuples?

  • Tuples use less memory than lists.
  • Useful for fixed data collections (like coordinates, RGB colors, etc.).
  • Can be used as keys in dictionaries (lists cannot).

Sets in Python

A set is an unordered collection of unique elements in Python. Sets are useful when you want to store multiple items, but don't care about their order and don't want any duplicates.

Creating a Set

You can create a set using curly braces {} or the set() constructor:

# Using curly braces
my_set = {1, 2, 3, 4}

# Using set() constructor
another_set = set([3, 4, 5, 6])

Key Features

  • Unordered: The elements have no defined order.
  • Unique: Duplicate elements are automatically removed.
  • Mutable: You can add or remove elements after creation.

Common Set Operations

a = {1, 2, 3}
b = {3, 4, 5}

# Union
print(a | b)  # {1, 2, 3, 4, 5}

# Intersection
print(a & b)  # {3}

# Difference
print(a - b)  # {1, 2}

Sets are great for removing duplicates from a list or performing mathematical set operations like union, intersection, and difference.

Python range() Function

The range() function in Python is used to generate a sequence of numbers. It's commonly used for looping a specific number of times in for loops.

Example: 
for i in range(5):
    print(i)
# Output: 0 1 2 3 4

You can specify a start, stop, and step value:
range(start, stop, step)

Example with start and step: 
for i in range(2, 10, 2):
    print(i)
# Output: 2 4 6 8

While Loop in Python

A while loop in Python repeatedly executes a block of code as long as a specified condition is True. It's useful when you don't know in advance how many times you need to repeat an action.

Example:
count = 1
while count <= 5:
    print("Count is:", count)
    count += 1

In this example, the loop prints numbers from 1 to 5. The loop stops when count becomes greater than 5.

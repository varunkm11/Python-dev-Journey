Hangman Game in Python

Welcome! This project will guide you through building a classic Hangman game using Python functions.

What you'll learn:
  • How to use functions to organize your code
  • Working with loops and conditionals
  • Handling user input and displaying output
  • Basic game logic and state management

Hangman is a fun word-guessing game. The player tries to guess a hidden word, one letter at a time, before running out of attempts. This project is perfect for beginners who want to practice Python programming in a hands-on way!

Python Data Types

In this lesson, we will learn about the different data types used in Python. Understanding data types is fundamental to programming because they determine the kind of data you can work with.

Common Python Data Types

  • int: Integer numbers, e.g., 5, -10.
  • float: Decimal numbers, e.g., 3.14, -0.001.
  • str: Text strings, e.g., "Hello", 'Python'.
  • bool: Boolean values, either True or False.
  • list: Ordered collection of items, e.g., [1, 2, 3].
  • dict: Key-value pairs, e.g., { "name": "Alice", "age": 25 }.
  • tuple: Immutable ordered collection, e.g., (1, 2, 3).
  • set: Unordered collection of unique items, e.g., {1, 2, 3}.

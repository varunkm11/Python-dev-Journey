Indentation In Python

Why is Indentation Important?

In Python, indentation is not just for readability—it's a core part of the language's syntax. Indentation defines blocks of code, such as the body of loops, functions, and conditionals. Incorrect indentation will cause errors.

Example:

if True:
    print("This is indented correctly")
    print("This line is also inside the if block")
print("This line is outside the if block")

Best Practices

  • Use 4 spaces per indentation level (the Python standard).
  • Be consistent—never mix tabs and spaces.
  • Use an editor that highlights indentation errors.

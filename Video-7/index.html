Python Variable Naming Rules

When creating variable names in Python, it's important to follow these rules and conventions:

  • Variable names must begin with a letter (A–Z, a–z) or an underscore (_).
  • Variable names can contain letters, digits (0–9), and underscores (_).
  • Variable names cannot begin with a digit.
  • Variable names are case-sensitive (name and Name are different).
  • Keywords (like if, while, class, etc.) cannot be used as variable names.

Examples of valid variable names:

  • my_var
  • _username
  • age2
  • totalAmount

Examples of invalid variable names:

  • 2ndPlace (starts with a digit)
  • user-name (hyphens not allowed)
  • class (reserved keyword)
